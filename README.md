# 📊 Sales & Customer Insights Dashboard

This project showcases an interactive dashboard developed using MicroStrategy.

## 🔍 Key Features
- Total Orders Analysis (45,000+ records)
- Customer Segmentation
- Global Sales Map Visualization
- Monthly Sales Trends

## 🛠 Tools Used
- MicroStrategy
- Data Visualization

## 📈 Outcome
Helps in understanding customer behavior and improving business decisions.
